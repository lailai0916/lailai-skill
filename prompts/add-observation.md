# Prompt: 记录一个新观察（不立刻写入规则）

复制以下内容给 AI，快速把一个观察记进 `observations.md`，不改正式规则。

---

请把下面这个观察追加到 `observations.md` 的 Pending Observations，**不要**改 `profile/` 或 `references/`。

观察：

【在此一句话描述你注意到的现象】

请按这个结构整理后追加（缺的信息向我确认，别编）：

```text
### YYYY-MM-DD 一句话标题

- 现象：……（尽量带原话 / 具体例子）
- 场景：……
- 推测：……（标明是推测）
- 稳定性：首次出现 / 第 N 次重现
- 处理：继续观察 / 待验证 / 建议沉淀到某文件
- 隐私检查：是否含隐私或他人敏感信息（应为"否"才保留）
```

规则：

- 观察不等于规则。只记录，不沉淀。
- 不写隐私、密钥、他人敏感信息（隐私检查为"是"则不记录，提醒我）。
- 不要把一次性现象说成稳定习惯。
- 若这条与某个已有规则冲突，指出来，但先不改规则，标注"待复核"。
