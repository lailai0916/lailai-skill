# Reference: Markdown 排版

## 目标

统一 Markdown 排版细节，符合 lailai 习惯。提炼自站点 `docs/project/guides/text.mdx`（Markdown / 标点 / 空格 / 数字 章节）与 `.claude/rules/writing-style.md`。

## 适用场景

- 写任何 Markdown / MDX 文档。
- 审查 Markdown 排版一致性。

> 用词见 [wording.md](wording.md)；公式见 [latex-math-style.md](latex-math-style.md)；站点专属（frontmatter、组件、题解模板）见 [docusaurus-style.md](docusaurus-style.md)。

## 文件

1. UTF-8 编码；文件名用 **kebab-case**（仅小写字母 + 数字，连字符分隔，不用大写 / 中文 / 空格 / 下划线 / 拼音）。
2. Markdown 用 `.md`、MDX 用 `.mdx`、纯文本 `.txt`、LaTeX 源 `.tex`。
3. 文件以 **1 个空行**结尾；**不连续用 2 个空行**；用 **2 空格**缩进（空行不缩进）。
4. MD 文件以 YAML frontmatter 开头：`title`（不写一级标题）、`description`（内容概述）。

## 标题

5. 用**二、三、四级**标题，避免五、六级，**不用一级标题**（正文最高 `##`）。
6. 不跳过标题等级；不用相同的上下级标题；不孤立标题编号（上级标题下不能只有 1 个下级）。
7. 标题是短名词短语，**结尾不加标点**。

## 强调与列表

8. **加粗**用 2 个星号 `**`（不用引号）；**斜体**用 1 个下划线 `_`（且尽量少用斜体）。
9. **无序列表**用连字符 `-`；有序列表用 `1.`（顺序有意义时）。
10. 列表标点：无序列表 / 任务列表每项末尾加**句号 `。`**；有序列表每项加**分号 `；`**，最后一项加**句号 `。`**。短列表可统一不加，但要一致。

## 标点（全角 / 半角）

11. 中文用**全角标点**，英文用**半角标点**。
12. 句末用 `。？！……`；不重复用标点（不写 `？？！！`）。
13. 引号用**直角引号「」『』**（不用弯引号 `""`），嵌套交替单双；**不用引号做强调**（强调用加粗）；书名用《》。
14. 中文省略号 `……`，英文 `...`。
15. 连接号区分：连字符 `-`（复合词）、短破折号 `–`（对等专名 / 区间范围）、长破折号 `——`（句内破折号）。人名音节分隔用间隔号 `·`。
16. 多处并列用顿号 `、`，两处并列用「和」/「或」；引号 / 书名号并列之间不加顿号。

## 空格

17. **中文与英文 / 数字之间加空格**；中文与**加粗 / 斜体 / 链接 / 代码 / 公式**之间加空格。
18. 数字与单位之间加空格；与简写计数 / `%` / `°` 之间不加。
19. 全角标点前后不加空格。行内代码 / 数学的句末标点放在外面。

## 链接、图片、代码、表格

20. 链接 `[描述性文本](url)`，相对路径省略 `./`；不用裸 URL（裸 URL 只在代码块里）。
21. 图片加替代文本 `![alt](url)`（站点图片惯例见 docusaurus-style）。
22. 行内代码块标语言标签（`cpp`，纯文本标 `text`）；C++ 遵循 [cpp-oi-style.md](cpp-oi-style.md)。
23. 表格居中对齐 `:-:`；分隔线用 3 个连字符 `---`。
24. 键盘按键用 `<kbd>`（`<kbd>⌘</kbd> + <kbd>C</kbd>`）；界面元素用方括号 `[]`。

## 数字与时间

25. 数字用半角阿拉伯数字，避免千分位；正式 / 约定俗成词用中文数字（「三局两胜」）。
26. 时间用 ISO 8601（`1969-07-20`、`20:17:40`）；货币用 ISO 4217（`CNY 6599`）。
27. 数量精确化：narrative 里的数量也包进行内数学（`$30$ 分`），见 [latex-math-style.md](latex-math-style.md)。

## 不推荐做法

- 一级标题、跳级、孤立编号、相同上下级标题。
- `*` / `+` 当无序列表符；弯引号 `""`；`...` 当中文省略号；`~` 当区间号。
- 中英 / 中数之间不空格；全角标点旁加空格。
- 裸 URL、缺 alt、代码块不标语言。

## Self-review Checklist

- [ ] 文件名 kebab-case，2 空格缩进，结尾 1 空行，无连续 2 空行？
- [ ] 正文最高 `##`，不跳级、不孤立编号、标题无句末标点？
- [ ] 加粗 `**`、斜体 `_`（少用）、无序列表 `-`，列表标点一致？
- [ ] 全角标点、直角引号「」、书名《》、省略号 `……`、破折号区分正确？
- [ ] 中英 / 中数 / 中-格式之间有空格，全角标点旁无空格？
- [ ] 链接描述性且省略 `./`，图片有 alt，代码块标语言？
- [ ] 表格 `:-:` 居中，分隔线 `---`，按键 `<kbd>`？
- [ ] 时间 ISO 8601，数量包进数学？
