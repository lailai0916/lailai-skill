# Reference: LaTeX 数学公式风格

## 目标

让数学公式排版符合 lailai 习惯。提炼自站点 `docs/project/guides/text.mdx`（LaTeX 章节）与 `docs/project/guides/latex.mdx`。

## 适用场景

- 任何含数学公式的内容（文章、笔记、题解、聊天）。
- 审查公式写法是否符合 lailai 习惯。

## 核心规则

### 分隔符与位置

1. **统一用 `$...$`（行内）与 `$$...$$`（行间）**，所有场景一致，不用 `\(...\)` / `\[...\]`。
2. 行间公式前后各 `$$` **单独成行**。**前一行散文以冒号 `：` 或句号 `。` 结尾**（不用逗号、不留空白无标点）；**公式末尾不带句读**——公式内容结尾**不写 `.`/`,`**（`\text{ans}=\dots\bigr)`，不是 `\bigr).`），闭合 `$$` 后也不接句号/逗号。注意 `\left\{…\right.` 的 `\right.` 是空定界符语法，那个 `.` 不算句读、保留。
3. 数学公式**不放在**标题、加粗、斜体、链接等格式里。
4. **不加多余空格**：运算符 / 关系符两侧不留空格（`a^2+b^2=c^2`，不是 `a^2 + b^2 = c^2`）；命令后紧贴下一个命令 / 括号（`\cos\theta`、`2\sqrt{5}`，不是 `\cos \theta`、`2 \sqrt{5}`）。**例外——内置函数后接数字时加一个空格**（`\sin 20`、`\sin 2\alpha`，不是 `\sin20`）。
   - **不手动调间距**：禁用一切显式间距命令 `\,` `\;` `\:` `\!` `\quad` `\qquad` `\ `（连写空格）`~`（不断行空格），交给 KaTeX 默认排版。括号紧贴内容：写 `c(f_{i,j})`，不写 `c(\,f_{i,\,j}\,)`。（注：上一条「命令后接数字加空格」用的是普通空格，不是这类间距命令。）
   - **函数名后不加花括号**：`\sin` `\cos` `\tan` `\ln` `\log` `\exp` `\min` `\max` `\lim` 等数学函数名后直接接参数——`\sin\theta`、`\cos(a+b)`、`\ln x`、`\log_2(n+1)`、`\exp(x^2)`、`\sin c`，不是 `\sin{\theta}`、`\ln{x+1}`、`\cos{\alpha}`（上标照常 `\sin^2\theta`）。但**结构命令仍用花括号**：`\frac{a}{b}`、`\sqrt{x}`、`\sqrt[n]{x}`、`\binom{n}{k}`、`\mathrm{d}x`、`\mathbb{R}`、`\operatorname{lcm}(a,b)`。

### 用数学语言，而非代码语言

5. 赋值用 `\to` 或 `\gets`，不用 `=`。
6. 取模用 `\bmod` 或 `\pmod`，不用 `%` 或 `\mod`。
7. 数列用 `a_i`，不用 `a[i]`。
8. **取整 / gcd / lcm 用规范记号，杜绝与中括号、圆括号撞义**：向下取整 `\lfloor x\rfloor`（**不写 `[x]`**），向上取整 `\lceil x\rceil`；最大公约数 `\gcd(a,b)`（**不写 `(a,b)`**），最小公倍数 `\operatorname{lcm}(a,b)`（**不写 `[a,b]`**）。
9. 位运算用 `\operatorname{and}` / `\operatorname{or}` / `\operatorname{xor}`。
10. 文本用 `\text`，字符串用 `\texttt`。
11. 未定义但约定俗成的函数用 `\operatorname`（如 `\operatorname{lca}`、`\operatorname{lcm}`）。

### 符号选择（用左、不用右）

12. **`\frac`** ← `\dfrac`。
13. 乘法 **`\times`** 或 **`\cdot`** ← `*`。
14. 绝对值 **`||`** ← `\lvert\rvert`。
15. 角度 **`\circ`** ← `\degree`。
16. 横向省略号 **`\dots`** ← `\cdots` / `\ldots`；纵向 `\vdots`；斜向 `\ddots`。
17. 空集 **`\varnothing`** ← `\emptyset`。集合用 **`\set{…}`** ← 裸 `\{…\}`（`\set{1,2,3}`、`\set{x\mid P}`、`\set{i}`）。例外：**数列** `\{a_n\}`（是 sequence 非集合，保留）、需**撑大**的 `\left\{…\right\}`（方程组 / 矩阵 / 含分数的集合，`\set` 不会自动撑大）。
18. 逻辑推出/反推/等价 **`\implies` / `\impliedby` / `\iff`** ← `\Rightarrow` / `\Leftarrow` / `\Leftrightarrow`。仅表示**逻辑**关系；赋值、变换仍用 `\to` / `\gets`（见 5）。否定的「不能推出」`\nRightarrow` 保留（无对应宏）。

### 格式细节

19. 变量名用**单个字母**加上下标，不用多字母（`a_i`，不是 `idx`）。
20. 物理单位用 `\mathrm`（`\mathrm{m/s^2}`）；罗马微分 `\mathrm{d}x`。
21. 特殊数集用 `\mathbb`（`\mathbb{R}`）。
22. 小括号若内含**非单行**内容（分数 `\frac`、根号 `\sqrt`、求和 `\sum` 等大高度元素）用 `\left( … \right)`；单行内容（如 `(a+b)`）不用。仅管小括号，中括号 `[…]` 不强制。
23. 复杂度用大 $O$ 记号，**不带常数**（`$O(n\log n)$`，不是 `$O(2n\log n)$`）。
24. 大数字用科学计数法。
25. 多步推导用 `\begin{aligned}`；分段用 `\begin{cases}`；矩阵用 `\begin{bmatrix}`。
26. 数字默认纯文本，`$...$` 只给真数学。散文里的年份 / 日期 / 分数 / 计数 / 百分比 / 度量是纯文本（`616 分`、`2 月 26 日`、`95%`、`17 km`、`20 世纪`）；只有作为公式 / 关系 / 变量 / 下标 / 运算符操作数的数才包（`$2^{10}=1024$`、`$3:1$`、`$\mu\pm 3\sigma$`、`$X\sim N(\mu,\sigma^2)$`）。
27. 范围：数学内用 `\sim`（`$1\sim n$`）；其余一律 en dash `–`，**两侧各留一个空格**（`1840 – 1842 年`、`13 – 19 岁`、`5 – 10 km`、`09:00 – 11:00`），任何地方不用波浪号字符（`～`/`〜`/`~`）表范围。对等专名（`牛顿–莱布尼茨`、`Bellman–Ford`）不是范围、保持紧贴。例外：版权声明的年份区间按惯例紧贴（`© 2021–2026`，与 en 源一致）。
28. **多行公式格式**（`\begin{...}\end{...}`）：每个分支单独成行、缩进 **2 个空格**；非最后一个分支以 ` \\` 结尾（反斜杠前一个空格，行距参数如 `\\[4pt]` 保留）；最后一个分支无 `\\`。对齐符 `&` **两侧各留一个空格**：`a & x>0`、`f(x) & =a`、`& =b`（ASCII 关系符 `=`/`<`/`>` 按规则 4 紧贴右操作数，无多余空格）；命令型关系符保留其后必要的一个空格（`& \equiv b`）。`\begin` / `\end` 各自成行。**优先保证渲染正确，而非格式绝对标准**——只动行内 `$...$` 的空格，多行换行只在行间 `$$...$$` 做（行内 `$...$` 不拆多行，remark-math 不解析跨行行内公式）。
29. **上下标顺序**：同一底数同时有上下标时，**下标在前、上标在后**——`\int_a^b`、`x_i^2`、`\chi_{0.05}^2`，不是 `\int^b_a`、`x^2_i`。
30. **单 token 上下标省略花括号**：上下标若只是**一个 token**（单字符或单个命令如 `\alpha`、`\infty`），省略花括号——`\int_a^b`、`\sum_{i=1}^n`、`\sin^2 x`、`x^\alpha`、`e^{-\lambda}` 里的 `e^\lambda`。但**多 token 必须保留**：`x^{-1}`、`x^{m+n}`、`a_{n_i}`、`a_{10}`、`a_{\mathrm{min}}`、`v_{\text{initial}}`。

## 示例

```latex
设 $S_k=\sum_{i=1}^{k}a_i$，则区间 $[l,r]$ 的和为 $S_r-S_{l-1}$。

$$
\begin{aligned}
  f(x) & =a \\
  & =b
\end{aligned}
$$
```

## Self-review Checklist

- [ ] 用 `$...$` / `$$...$$`，行间 `$$` 单独成行，公式不在标题 / 加粗里？
- [ ] 用数学语言（`\to` 赋值、`\bmod` 取模、`a_i` 而非 `a[i]`、`\lfloor\rfloor`）？
- [ ] 符号选对（`\frac` 非 `\dfrac`、`\dots` 非 `\cdots`、`\varnothing` 非 `\emptyset`、`\times` 非 `*`）？
- [ ] 变量单字母、复杂度大 $O$ 不带常数、单位 `\mathrm`、数集 `\mathbb`？
- [ ] 运算符 / 命令两侧无多余空格（`a+b`、`\cos\theta`）？内置函数后接数字加了一个空格（`\sin 20`）？三角函数后无无意义花括号（`\sin\theta` 非 `\sin{\theta}`）？含分数 / 根号的小括号用了 `\left\right`？没用 `\,` `\;` `\quad` 等手动间距命令？
- [ ] 多行公式分支缩进 2 空格、` \\` 行尾（前一空格、最后分支无）、`&` 两侧各一空格且关系符紧贴操作数（`& =a`）？
- [ ] 散文数字是纯文本、`$...$` 只给真数学？范围数学内 `\sim`、其余 en dash `–`（两侧加空格）、无波浪号？
