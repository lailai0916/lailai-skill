# Reference: 用词规范

## 目标

统一中文用词，匹配 lailai 的习惯用字。提炼自站点 `docs/project/guides/wording.mdx`（lailai 自己的《用词规范表》）。

## 适用场景

- 写或润色任何中文内容。
- 审查用词是否符合 lailai 习惯。

> 这是高频、强信号的规则。用错字、用了 lailai 反感的词（如「显然」「易得」），即使内容对，也立刻不像 lailai。

## 通用

- 正确区分「的」「地」「得」。
- 正确区分「他」「她」「它」。
- 正确区分「与」「和」「或」。

## 错别字（用左、不用右）

- 「模板」← 模版
- 「登录」← 登陆
- 「账号」← 帐号
- 「其他」← 其它

## 近义词取舍（用左、不用右）

- 「你」← 您
- 「仅」← 只
- 「若」← 如果
- 「必然」← 必定
- 「然后」← 接下来
- 「包含」← 含有
- 「相互」← 互相
- 「依此类推」← 以此类推
- 标题用「关于」，正文用「有关」。
- 用「例如」和「示例」。

## 强反感（禁用）

- **避免使用「显然」「易得」「不难发现」**。这类词把推理跳过去，既不严谨也是典型 AI 腔 / 装腔。要么写清楚，要么不写。

## 专业术语（用左、不用右）

- 「连通」← 联通
- 「扩展」← 拓展
- 「源代码」← 源码

## Self-review Checklist

- [ ] 「的 / 地 / 得」「他 / 她 / 它」「与 / 和 / 或」用对？
- [ ] 没有「模版 / 登陆 / 帐号 / 其它」这类错字？
- [ ] 近义词按表取舍（你 / 仅 / 若 / 然后 / 包含 / 相互……）？
- [ ] 没有「显然」「易得」「不难发现」？
- [ ] 术语用「连通 / 扩展 / 源代码」？
